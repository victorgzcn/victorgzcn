Example Workflow
# Clone a repository
git clone https://github.com/user/project.git
cd project

# Create a branch
git checkout -b fix-bug

# Make changes, then save them
git add .
git commit -m "Fix critical bug"

# Push to GitHub
git push origin fix-bug

# Create a Pull Request on GitHub's website
